# 🍎 OrchardLink

🚀 Full-stack platform connecting apple farmers in Kashmir directly with consumers.

👉 Adopt a tree → Track growth → Get fresh apples

---

## 🔗 Live Demo:https://orchardlink.vercel.app/


## ✨ Key Features

* 🌳 Tree adoption system
* 👨‍🌾 Farmer dashboard (add trees)
* 👤 User dashboard (track adopted trees)
* 📸 Image upload via Cloudinary
* 🔐 Role-based authentication
